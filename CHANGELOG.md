# Changelog

## 2.1.0 — 2025-12-26
- Автообновление через GitHub, кнопка «Обновить» в панели.
- Чистый светлый UI, новые кнопки/табы, анимации наведения.
- Гайд закрывается по фону и Esc; аккуратная верстка попапа.
- Автоопределение страницы и автопереключение вкладки (Журнал/Баннер/Акция).
- Цвет дисклеймера баннера по бренду; корректный тип статьи («Новость»).
- Тумблер «Свернуть/Развернуть» меняет подпись по состоянию.

## 2.0.0 — 2025-12-18
- Прод-рефакторинг, DEBUG-флаг, защита повторной инъекции.
- Гайд + модалка, перетаскиваемая панель с сохранением позиции.
- Самопроверка страницы, стабильные парсеры ТЗ (Журнал/Баннер/Акция), улучшенные логи.
